# system-rejestracji


## Installation

Install the required packages with:

```sh
pip install -r requirements.txt
```

## Running the Server

In `/backend` directory, run:

On windows:
```sh
py ./manage.py runserver
```

On Linux:
```sh
python3 manage.py runserver
```

Then open `http://127.0.0.1:8000/` in your browser.
